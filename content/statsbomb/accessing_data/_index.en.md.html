Statsbomb Website

Statsbomb kindly provide a range of data sets at no cost. An overview of the available data can be found here: Free Data. The code below can be ran in R to see which specific competitions and data sets you can access.

#Required packages
library(tidyverse)
library(StatsBombR)
library(dplyr)
#Loads competitions which are available
Comp <- FreeCompetitions() #this gives you the competitions ids used below
view(Comp)
